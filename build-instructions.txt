1) Create a directory in your Windows machine 

2) Download the following files in directory you just created:
   - source code (Inspector.java, Notepad.java and manifest) 
   - build.txt 

3) Rename build.txt as build.bat (Google mail doesnt allow to attach *.zip or *.bat files)

4) Open command prompt and go to directory where files are stored 

5) Execute build.bat --> Reporter.jar will be created 

==============================================================

6) Copy your CSV input file into the same directory 

7) Execute Reporter.jar
      java -jar Reporter.jar <input-csv-file> <output-csv-file>
   Replace the values of <input-csv-file> <output-csv-file>
